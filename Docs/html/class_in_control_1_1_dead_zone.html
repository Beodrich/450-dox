static

Apply deadzones to a vector, but processes the deadzones on the vector as radii on a circle.

This works well for analog sticks as they need to operate within a minimum and maximum radius.

Parameters
xThe x component of the vector to process.
yThe y component of the vector to process.
lowerDeadZoneThe lower dead zone to apply (e.g. 0.2f)
upperDeadZoneThe upper dead zone to apply (e.g. 0.9f)

static

Same as SeparateNotNormalized, but the resulting vector is normalized so the result lands on the edge of a circle with a radius of 1.0f.

This is useful for d-pad controls where the input snaps to 8 directions, making the result interchangable with the output for analog sticks.

Parameters
xThe x component of the vector to process.
yThe y component of the vector to process.
lowerDeadZoneThe lower dead zone to apply (e.g. 0.2f)
upperDeadZoneThe upper dead zone to apply (e.g. 0.9f)

Apply deadzones to a vector, but processing the x and y components separately.

The resulting vector is not normalized, so the result falls within a rectangle. This is not usually the kind of deadzone you want and may give undesired results.

Parameters
xThe x component of the vector to process.
yThe y component of the vector to process.
lowerDeadZoneThe lower dead zone to apply (e.g. 0.2f)
upperDeadZoneThe upper dead zone to apply (e.g. 0.9f)
